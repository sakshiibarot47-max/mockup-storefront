# 🛍️ Mockup Storefront

A modern, responsive e-commerce mockup application built with React and Vite. This project showcases a clean user interface, dynamic product listings, and an interactive shopping cart experience.

## 📊 Dashboard Preview
![Storefront Dashboard](dashboard1.png)

## 🚀 Features
* **Dynamic Product Catalog:** Sleek grid layout displaying items with clean image placements, pricing, and descriptions.
* **Interactive Cart Engine:** Live add-to-cart functionality updating totals and quantities on the fly.
* **Responsive Design:** Fully optimized for seamless browsing across desktop, tablet, and mobile displays.
